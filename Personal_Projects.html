Personal Projects

Project Name Description
Point and Shoot Camera First project I completed with a Raspberry Pi. The project used a Raspberry Pi 3, Raspberry Pi Camera Module, and a Adafruit 16x2 lcd and keypad kit. The program was written completely in python with an option to shutdown the whole system. The camera is able to take pictures and record video with an option to increase or decrease the length.
Camera Effect Selector Simple program that allows the user to interact with the python shell giving the user the option to take a photo with an effect. The program shows a preview image of the photo before taking the photo to confirm to the user that they want to take the photo.
